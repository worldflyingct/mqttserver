## 使用方法
go get -v -u github.com/worldflyingct/mqttserver  
## 产品说明
GO语言版本并非完整的运行程序，仅仅是一个运行库，方便给自己开发的应用添加mqtt功能。  
C语言版本的是完整的运行程序，所有的io操作都是异步非阻塞io，使用了epoll技术实现高并发。
